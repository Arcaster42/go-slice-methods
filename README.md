# Go Slice Methods
This package adds functions for working with slices similar to that of other languages. It aims to replicate many of the functionalities included in the JavaScript `Array` prototype.

## Features
- Filter by values
- Filter by function

## Features
### Primitive Filters
- `FilterInt`
- `FilterString`
- `FilterFloat32`
- `FilterFloat64`

### Logical Filters
- `FilterIntByFunc`
- `FilterStringByFunc`
- `FilterFloat32ByFunc`
- `FilterFloat64ByFunc`

## Upcoming Features
- `map`
- `reduce`
